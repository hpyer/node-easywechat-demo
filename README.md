### 使用

1、克隆项目到本地

`git clone git@github.com:hpyer/node-easywechat-demo.git`

2、进入项目目录

`cd node-easywechat-demo`

3、安装依赖包

`npm install`

4、配置 config 目录下的`OfficialAccount.js`, `Payment.js`两个文件

5、运行项目

`npm run koa2`
